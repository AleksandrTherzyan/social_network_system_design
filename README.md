# System Design социальной сети для курса по System Design

### Функциональные требования:

- Публикация постов из путешествий с фотографиями, небольшим описанием и привязкой к конкретному месту путешествия;
- Оценка и комментарии постов других путешественников;
- Подписка на других путешественников, чтобы следить за их активностью;
- Поиск популярных мест для путешествий и просмотр постов с этих мест;
- Просмотр ленты других путешественников и ленты пользователя, основанной на подписках в обратном хронологическом порядке;
- Каталог экскурсий по направлениям(рейтинг, комментарии) с возможностью покупки
- Интеграция с платежными системами
- Адаптивность под разные устройства

### Нефункциональные требования:
- DAU `10 000 000`
- Регион использования: СНГ
- Доступность критического функционала: `99.999` (Просмотр ленты пользователя и ленты других путешественников, публикация постов, просмотр постов по местам, поиск популярных мест, каталог экскурсий, покупка экскурсий)
- Доступность не критического функционала: `99.99` (Лайки и комментарии к постам или в каталоге экскурсий, подписки/отписки, оценки)
- Сезонные всплески нагрузки (Сезонный множитель ×2.5 летом и в новогоднии праздники)
- Данные храним всегда
- Должны быть настроены бекапы данных для каждого узла
- CDN для статики
- Горизонтальное масштабирование (добавление новых серверов в кластер, использование stateless сервисов, балансировщики,)
- Будет дополнено по мере прохождения курса :)

#### Средние дневные действия пользователя:
| Действие | количество |
|----------|------------------------------|
| Публикация постов | 0.01 (1 пост на 100 человек) |
| Комментарии к постам | 0.05 (1 коммент на 20 человек) |
| Лайки | 0.5 (каждый второй ставит лайк) |
| Подписки / отписки | 0.01 |
| Просмотры ленты | 5 |
| Просмотры постов по местам | 2 |
| Поиск популярных мест | 0.2 |
| Просмотры каталога экскурсий | 1 |

| Комментарии / рейтинг экскурсий | 0.02 |

  Пиковую нагрузку будем считать `x5` от средней

## Расчет нагрузки

#### Публикация постов из путешествий
- Средний RPS: `100 000 / 86 400 ~= 1.16`
- Пиковый RPS: `~= 6`

#### Лайки и комментарии постов
- Лайк средний RPS: = `5 000 000 / 86 400 ~= 58`
- Лайк пиковый RPS: `~= 290`
- Создание комментария средний RPS: = `500 000 / 86 400 ~= 5.8`
- Создание комментария пиковый RPS: `~= 29`

#### Подписка/отписка на других путешественников
- Cредний RPS = `100 000 / 86 400 ~= 1.16`
- Пиковый RPS = `~= 6`

#### Поиск и просмотр популярных мест
- Поиск средний RPS = `2 000 000 / 86 400 ~= 23 RPS`
- Поиск пиковый  RPS `~= 115 RPS`
- Просмотр средний RPS = `20 000 000 / 86 400 ~= 231 RPS`
- Просмотр пиковый RPS `~= 1 160 RPS`

#### Лента других путешественников и лента пользователя
- Средний RPS = `50 000 000 / 86 400 ~= 579 RPS`
- Пиковый RPS `~= 2 900 RPS`

#### Каталог экскурсий по направлениям, рейтинг, комментарии, покупка
- Просмотр каталога средний RPS = `10 000 000 / 86 400 ~= 116 RPS`
- Просмотр каталога пиковый RPS `~= 580 RPS`
- Попкупка средний RPS = `10 000 / 86 400 ~= 0.12 RPS`
- Попкупка пиковый RPS `~= 0.6 RPS`
- Комментарии/рейтинг средний RPS = `200 000 / 86 400 ~= 2.3 RPS`
- Комментарии/рейтинг пиковый RPS `~= 13 RPS`
